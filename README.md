\🛡️ EDR Lite – Endpoint Detection & Response System

A lightweight Endpoint Detection & Response (EDR) system built using FastAPI, React, PostgreSQL, and Docker.
It collects system telemetry from an agent, stores it in a database, and visualizes security insights through a web dashboard.

⚙️ Tech Stack
Backend: FastAPI (Python)
Frontend: React (Vite + Nginx)
Database: PostgreSQL
ORM: SQLAlchemy
Containerization: Docker & Docker Compose
🧠 System Architecture
Agent → FastAPI Backend → PostgreSQL → React Dashboard
The agent collects system-level data (CPU, processes, file hashes, ports)
The backend processes and stores telemetry data
The database persists all security events
The frontend dashboard visualizes everything in real time
🚀 Key Features
📊 System Monitoring
CPU, memory, and disk usage tracking
Process monitoring
Open port detection
🔐 Security Monitoring
File integrity monitoring using hash comparison
Failed login tracking
Basic alert generation system
📡 Data Pipeline
Agent → API ingestion → PostgreSQL storage → Dashboard visualization
🐳 Docker Deployment

The entire system is containerized:

PostgreSQL database container
FastAPI backend container
React frontend served via Nginx
Run the system:
docker compose up --build
Access:
Dashboard: http://localhost:3000
API Docs: http://localhost:8000/docs

🚨 Current Limitations

Agent must be manually started
Basic authentication only (JWT minimal setup)
No advanced threat detection logic yet
